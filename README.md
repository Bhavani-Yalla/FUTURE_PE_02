# AI Content Marketing Using UGC Ads

## Project Overview

This project was created as part of the Future Interns Prompt Engineering Task 2 (2026).

The objective was to design a prompt system that generates UGC-style ad content for a real business and create marketing content that can be used for Instagram Reels and social media advertising.

## Business Selected

Bright Future Academy, Vijayawada

Bright Future Academy provides coaching for:

* IIT-JEE
* NEET
* EAMCET

## Prompt Used

Create a UGC-style Instagram Reel ad for Bright Future Academy.

Framework:
Hook → Problem → Solution → Result → CTA

Length:
25–30 seconds

Include:

* Scene descriptions
* Dialogue
* On-screen text

Tone:
Authentic, relatable, conversational.

## Generated Output

The prompt generated a UGC-style Instagram Reel script consisting of:

1. Hook
2. Problem
3. Solution
4. Result
5. Call To Action (CTA)

The script was designed to connect with students preparing for competitive exams and encourage them to book a free demo class.

## Tools Used

* ChatGPT
* Pika
* Canva
* GitHub

## Deliverables

* UGC Ad Prompt
* UGC Ad Script
* Promotional Banner
* Reel Assets
* Screenshots
* GitHub Documentation

## Outcome

Successfully created a complete UGC Ad Content Pack for Bright Future Academy that can be used for Instagram Reels, social media marketing, and paid advertising campaigns.
